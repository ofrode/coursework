# PsychologicalTests

Приложение для проведения психологических тестов на Qt/C++.

## Структура проекта

```
├── src/           # Исходные файлы (.cpp)
│   ├── core/      # Основные классы
│   ├── ui/        # Классы интерфейса
│   └── utils/     # Вспомогательные классы
└── hdr/           # Заголовочные файлы (.h)
    ├── core/      # Основные классы
    ├── ui/        # Классы интерфейса
    └── utils/     # Вспомогательные классы
```

## Основные классы

- **Test** — класс психологического теста
- **Question** — класс вопроса
- **Answer** — класс ответа
- **TestResult** — результат прохождения теста
- **Statistics** — статистика по тесту
- **DataManager** — менеджер данных (связывает Question и Statistics)

## Сборка

```bash
mkdir build && cd build
cmake ..
make
```

## Запуск

```bash
./build/PsychologicalTests
```
